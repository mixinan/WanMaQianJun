写此项目的**[目的]()**：
> 1. 很喜欢Twitter的标题栏，想模仿一下。
> 2. 对微信的导航图标颜色渐变很好奇，想实现一下。
> 3. 想练习`ViewPager + Fragment`、 `SwipeRefreshLayout`的使用。
